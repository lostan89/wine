# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".


### Как установить

**Python3.13** должен быть уже установлен. 

Скачайте код с **GitHub**. Затем установите зависимости.

```python
pip install -r requirements.txt
```
## Запуск

- Запустите сайт командой
```python
python3 main.py
```
- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).

### Примечание:

Программа загружает на сайт цены и ассортимент напитков из таблицы.

В репозитории есть готовый файл для редактирования (`wine3.xlsx`)

Файл таблицы по умолчанию должен находиться в корне программы, но вы можете использовать свой файл, добавив путь к нему в переменную окружения **.env** (`EXCEL_FILE_PATH=`)
